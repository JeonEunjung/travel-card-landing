# MOIN Landing Page

## Design Skills (Supanova Design Skill)

이 프로젝트는 Supanova Design Skill을 사용하여 프리미엄 랜딩페이지를 생성/개선합니다.

스킬 파일들은 `.claude/` 디렉토리에 위치합니다:

- `.claude/taste-skill.md` — 메인 디자인 엔진 (프리미엄 랜딩페이지 생성)
- `.claude/redesign-skill.md` — 기존 페이지 리디자인/업그레이드
- `.claude/soft-skill.md` — $150k 에이전시 퀄리티 미학 기준
- `.claude/output-skill.md` — AI 출력 생략 방지, 완전한 HTML 출력 강제

### 사용 방법
- 새 랜딩페이지 생성 시: taste-skill + output-skill 참조
- 기존 페이지 개선 시: redesign-skill 참조
- 최고 퀄리티 필요 시: taste-skill + soft-skill + output-skill 모두 참조

### 기술 스택
- Standalone HTML (단일 파일)
- Tailwind CSS (CDN)
- Pretendard 폰트 (한국어)
- Iconify Solar 아이콘
